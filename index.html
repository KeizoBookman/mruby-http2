HTTP2 Module for mruby

HTTP2 module for mruby. You can access Web site using HTTP/2.0 protocol from mruby applications or devices with mruby.

  • HTTP/2.0

    HTTP/2.0 is a replacement for how HTTP is expressed ¡Èon the wire.¡É It is not a ground-up rewrite of the protocol; HTTP methods, status codes and semantics will be the same, and it should be possible to use the same APIs as HTTP/1.x (possibly with some small additions) to represent the protocol.

install by mrbgems

  • Download
git clone https://github.com/mruby/mruby.git
git clone https://github.com/matsumoto-r/mruby-http2.git
  • nghttp2 build
cd mruby-http2
git submodule init
git submodule update
cd src/nghttp2
autoreconf -i
automake
autoconf
./configure
make
cd ../../../
  • add conf.gem line to build_config.rb 
cd mruby

MRuby::Build.new do |conf|

  # ... (snip) ...

  conf.gem '../mruby-http2'
end
  • build
rake
